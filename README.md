#### 🔹 [ImprovedBoardVision](https://github.com/hraskin/ImprovedBoardVision)

A tool designed to help visually impaired students follow lessons more easily.
It captures board content and converts it into audio using accessible devices (smartphone + laptop).

**Key features:**

* 📸 Capture board content via screenshots
* 🎤 Voice commands & keyboard shortcuts
* 🔊 Audio playback for later review
* 💻 Works with common devices (iPhone + MacBook)
